# MedSafe Open Source

## Cronograma de Desarrollo (12 Meses)

### Fase 1: Fundación y Gobernanza (Meses 1-2)
- Creación del repositorio (GitHub/GitLab), elección de licencia (MIT/Apache 2.0), Manifiesto del Proyecto.
- Squads:
  - Hardware: PCB y Modelado 3D
  - Software: Backend (Gov.br) y App móvil
  - IA/Datos: Modelos de interacción medicamentosa
- Hito: ReadMe y directrices de contribución.

### Fase 2: MVP 1.0 (Meses 3-6)
- STL 3D, firmware ESP32, integración App Beta.
- Hardware Freeze 1.0.
- Hito: Video "Hello World".

### Fase 3: Inteligencia y Conectividad (Meses 7-9)
- IA de barrera, reconocimiento facial, pruebas de ciberseguridad.
- Hito: White Paper técnico.

### Fase 4: Piloto y Homologación (Meses 10-12)
- Prototipos, recopilación UX, ajustes finales.
- Hito: Evento de Lanzamiento Open Source v1.0.

---

## Estructura de Colaboración

- **Gobernanza:** Comité internacional + squads locales.  
- **Squads:** Hardware, Software, IA/Datos.  
- **Herramientas:** GitHub/GitLab, licencia abierta, documentación.  
- **Comunidad:** Pull Requests, revisión por pares, foros.  
- **Hitos:** Manifiesto, White Paper, Evento v1.0.
